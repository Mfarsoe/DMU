Velkommen til DocJava

Dette er et redesign af DocJava.dk, skabt med fokus på semantisk HTML og moderne webstandarder. Siden er optimeret til undervisning i Java-programmering.

Hvorfor lære Java?

Java er et af verdens mest populære programmeringssprog. Det er platformsuafhængigt (Write Once, Run Anywhere), objektorienteret og har et kæmpe økosystem af biblioteker.

Installation & Miljø

For at komme i gang med Java skal du installere et Java Development Kit (JDK).

JDK vs JRE

  • JRE (Java Runtime Environment): Bruges kun til at køre Java-programmer.
  • JDK (Java Development Kit): Bruges til at udvikle og køre Java-programmer.
