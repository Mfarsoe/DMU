HTML og CSS Introduktion

Formålet er at danne sig en grundlæggende viden inden for HTML og CSS.

HTML Opgaver

Hvad er HTML en forkortelse for, og hvor stammer HTML fra?

HTML er en forkortelse for HyperText Markup Language. HyperText refererer til tekst med links, som forbinder dokumenter. Dette er grundlaget for internettet. Markup Language betyder et mærkningssprog, der bruges til at strukturere og formatere indholdet i et dokument. Dette kan være tekst, billeder og links. HTML beskriver, hvordan en internetbrowser skal vise indhold ved hjælp af tags.

HTML blev udviklet i begyndelsen af 1990'erne af Tim Berners-Lee, computerforsker og fysiker. Målet var at skabe en nem måde at dele og sammenkoble information på mellem forskere over internettet. Først version var i 1991, og var også grundlaget for World Wide Web (Internettet).

Hvad er historien bag, og forskellen på HTML4, XHTML, og HTML5?

HTML4 blev udgivet i 1997 af W3C og var i mange år den mest brugte version af HTML på nettet. Der var fokus på struktur og ikke så meget på design og interaktivitet. Introducerede formularer, tabeller, og grundlæggende struktur-elementer. HTML4 var med forkus på vidensdeling.

XHTML blev introduceret i 2000 som et forsøg på at forbedre HTML4 ved at gøre det mere strukturelt strengt og XML-kompatibelt. XHTML står for Extensible HyperText Markup Language, og er en version af HTML som skal følge XML-regler:

  • Alle tags skal lukkes korrekt.
  • Alt skal være med små bogstaver.
  • Attributter skal have citationstegn.
Til gengæld er det mere forudsigelig for computeren og browseren.

HTML5 blev først gang udgivet i 2014. Udviklet af WHATWG (Web Hypertext Application Technology Working Group) og senere standardiseret af W3C. HTML5 er løbende blevet opdateret og er pt den aktuelle standard. I HTML 5 er der fokus på multimedie, interaktivitet og semantik. Der er indbygget understøttelse af video, lyd, grafik, geolocation, offline-funktioner m.m.

Hvad er <div> og hvordan bruges class?

<div> er et blok-element i HTML, som bruges til at gruppere andre HTML-elementer. Det har ingen visuel effekt i sig selv, men bruges til at strukturere layout og indhold på en webside.

Ved at bruge class kan du give din <div> (eller andre HTML-elementer) et navn, som du senere kan bruge til at tilføje styling via CSS eller manipulere med JavaScript.

Eksempel på brug af <div> med class:

<div class="header">
  <h2>Dette er headeren</h2>
</div>

<div class="columns">
  <div class="left">
    <h3>Venstre kolonne</h3>
    <p>Indhold i venstre kolonne.</p>
  </div>
  <div class="right">
    <h3>Højre kolonne</h3>
    <p>Indhold i højre kolonne.</p>
  </div>
</div>

<div class="footer">
  <p>Dette er footeren</p>
</div>

Her bruges forskellige class-navne: header, columns, left, right og footer. Disse kan styles via CSS.

CSS Opgaver

Hvad er CSS?

CSS står for Cascading Style Sheets, og det bruges til at styre udseendet og layoutet af HTML-elementer på en webside. CSS blev udviklet af W3C (World Wide Web Consortium) i midten af 1990’erne som en måde at adskille indhold (HTML) fra præsentation (stil og design).

Hvad er selectors?

En selector i CSS er den del, der udvælger hvilket HTML-element eller hvilke elementer, reglerne skal gælde for. De bruges til at koble CSS-regler sammen med specifikke elementer.

Eksempler på selectors:

  • p - vælger alle <p>-tags
  • .menu - vælger alle elementer med class="menu"
  • #footer - vælger elementet med id="footer"
  • h1, h2 - vælger både h1 og h2

Hvilke egenskaber kan man angive for tekst?

Med CSS kan man ændre tekstens udseende med følgende egenskaber:

  • font-family - hvilken skrifttype der bruges
  • font-size - hvor stor teksten er
  • font-weight - hvor tyk/fed teksten er
  • font-style - fx italic (kursiv)
  • text-align - hvordan teksten skal placeres (venstre, højre, center)
  • line-height - linjeafstand
  • letter-spacing - afstand mellem bogstaver

Hvordan angives en fonts egenskaber?

Man kan bruge både font-forkortelsen eller skrive egenskaber enkeltvis:

p {
  font-family: Arial, sans-serif;
  font-size: 16px;
  font-weight: bold;
  font-style: italic;
}

Det angiver skrifttype, størrelse, fed tekst og kursiv for et afsnit.

Hvordan angives farver?

Farver angives typisk med:

  • navne: red, blue, black
  • hex-koder: #ff0000 (rød), #000000 (sort)
  • RGB: rgb(255, 0, 0)
  • HSL: hsl(0, 100%, 50%)
body {
  color: #333333;
  background-color: #f9f9f9;
}

Hvordan angives størrelser?

Man kan angive størrelser i forskellige enheder:

  • px - pixels (fast størrelse)
  • em - relativ til elementets skrifttype
  • rem - relativ til root element (html)
  • % - relativ til forælderens størrelse
h1 {
  font-size: 2em;
}

p {
  width: 80%;
}

Hvad er padding og margin?

Padding er den indvendige afstand mellem indholdet og kanten af et element.

Margin er den udvendige afstand mellem elementet og det omkringliggende indhold.

div {
  padding: 20px;
  margin: 30px;
}

Ovenstående betyder, at der er 20px luft inde i boksen og 30px luft udenom boksen.

Flere vigtige CSS properties

  • opacity: Gennemsigtighed på et element.
    opacity: 0.5; gør elementet halvgennemsigtigt.
  • visibility: Skjuler eller viser et element, men bevarer pladsen.
    visibility: hidden;
  • width: Bredden på et element.
    width: 300px; eller width: 50%;
  • height: Højden på et element.
    height: 200px;
  • top / left: Bruges til at placere elementer, når man bruger positionering.
    top: 20px;, left: 10px;

Hvordan kan man anvende CSS i HTML?

Der er tre måder at tilføje CSS til en HTML-side:

  1. Inline CSS - direkte i et HTML-element:
    <p style="color: red;">Rød tekst</p>
  2. Internal CSS - i et <style>-tag i <head>-sektionen:
    <style>
  p { color: blue; }
</style>
  3. External CSS - i en separat .css-fil:
    <link rel="stylesheet" href="style.css">

Positionering i CSS

Standard positionering i HTML er static. Det betyder, at elementer vises i den rækkefølge, de kommer i dokumentet.

Andre positionstyper:

  • relative - element flyttes i forhold til sin normale placering
  • absolute - placeres i forhold til nærmeste positionerede forælder
  • fixed - fast placering i forhold til vinduet (fx sticky menu)
  • sticky - opfører sig som relative, men "klistrer" ved scrolling
.box {
  position: absolute;
  top: 50px;
  left: 100px;
}

Hvordan fungerer CSS Flexbox?

Flexbox er en layoutmetode til at arrangere elementer i én række eller én kolonne. Den er god til at fordele plads og justere elementer.

.container {
  display: flex;
  justify-content: space-between;
  align-items: center;
}
  • justify-content styrer vandret placering
  • align-items styrer lodret placering

Hvordan fungerer CSS Grid?

Grid er en mere avanceret layoutmetode end Flexbox, og bruges til at lave todimensionelle layout – både rækker og kolonner.

.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr;
  gap: 20px;
}

Det skaber to kolonner af samme størrelse med 20px afstand imellem.

Hvad er Bootstrap?

Bootstrap er et populært CSS-framework udviklet af Twitter. Det indeholder færdiglavede klasser og komponenter, der gør det nemt at lave responsive og moderne websites uden selv at kode alt CSS fra bunden.

Med Bootstrap kan man:

  • Brug færdige layouts og grid-systemer
  • Brug knapper, formularer, tabeller og kort uden ekstra CSS
  • Tilpasse designet til mobil og desktop automatisk (responsive design)
  • Tilføje komponenter som modals, navigation og sliders

Eksempel på en Bootstrap-knap:


<button class="btn btn-primary">Klik her</button>

Bootstrap kræver at du inkluderer et link til frameworket i <head>-sektionen:

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
